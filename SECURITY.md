# Security Policy

## Supported Versions

Paste Wrong File Blocker is in early development. All versions from `0.0.1` onward receive security attention as needed.

## Reporting a Vulnerability

If you discover a security issue, please report it privately.

Email: security@missioncriticalsoftware.dev

Please include:

- A clear description of the issue
- Steps to reproduce
- Expected vs. actual behavior
- Any relevant logs or environment details

We will acknowledge receipt within 48 hours and provide a timeline for resolution.

## Security Philosophy

Paste Wrong File Blocker follows a deterministic safety model:

- Minimal attack surface
- No network access
- No external dependencies
- No dynamic code execution
- No telemetry
- No user data collection

The extension operates entirely within the local VS Code environment.

## Disclosure

We follow responsible disclosure practices.
Fixes will be released promptly and documented in the `CHANGELOG.md`.
