# Support Policy

Paste Wrong File Blocker is maintained as an operator‑grade, deterministic tool.  
Support is provided on a best‑effort basis with clear boundaries and expectations.

---

## 1. Where to Get Help

### **Primary Support Channel**
GitHub Issues:  
https://github.com/brotherbill/paste-wrong-file-blocker/issues

Use this for:

- Bug reports  
- Validation rule inconsistencies  
- Unexpected behavior  
- Documentation corrections  
- Feature requests (must align with deterministic doctrine)

### **Security Issues**
Do **not** open a public issue.  
Follow the instructions in `SECURITY.md`.

---

## 2. What Is Supported

The following areas are fully supported:

- Paste validation logic  
- Header/footer rule enforcement  
- Modal error code behavior  
- Extension activation and command execution  
- Deterministic behavior across platforms  
- Documentation accuracy  
- Packaging and VSIX integrity

---

## 3. What Is Not Supported

The following are out of scope:

- Custom validation rules not aligned with project doctrine  
- Integration with other extensions  
- VS Code bugs or platform issues  
- Git configuration or local filesystem permissions  
- Clipboard behavior outside VS Code  
- Modified or unofficial builds  
- Telemetry, analytics, or network features (the extension has none)

---

## 4. How to File a High‑Quality Issue

Include:

1. **VS Code version**  
2. **Extension version**  
3. **Operating system**  
4. **Exact clipboard content** (sanitized if necessary)  
5. **Expected behavior**  
6. **Actual behavior**  
7. **Steps to reproduce**  
8. **Screenshots of modal error messages**  

Issues missing required information may be closed.

---

## 5. Response Expectations

- Issues are typically acknowledged within **72 hours**  
- Fixes are prioritized based on severity and reproducibility  
- Deterministic behavior regressions receive highest priority  
- Feature requests are evaluated for doctrinal alignment

---

## 6. Long‑Term Support

This project is maintained actively.  
Only the latest published version receives updates.
